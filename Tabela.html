Tabelas HTML

A tabela HTML pode ser utilizada para quando o desenvolvedor web desejar organizar seus dados em linhas e colunas.

Definindo tabela:

O conceito de tabela em HTML, consiste em células de tabela dentro de linhas e colunas. Para tanto é necessário utilizar o atributo <table> para criar uma tabela.


Exemplo:


Resultado:


Compras Quantidade
Arroz 5
Feijão 10

Explicação:


Tag início Conteudo Tag final
<td> É o conteúdo da célula da tabela </td>
<tr> Representa as linhas da tabela </tr>
<th> Representa o cabeçalho da tabela </th>
