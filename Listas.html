Listas HTML

As listas HTML foram criadas para permitir aos desenvolvedores a agruparem itens relacionados em listas. Tais listas podem ser classificadas como ordenadas, não-ordenadas, descritivas entre outras...

Lista não-ordenada HTML:

Uma lista não-ordenada inicia com a tag <ul>. Para classificar cada item da lista usa-se a tag <li>. Os itens dessa lista serão exibidos com marcadores como padrão. Esses marcadores são uma espécie de círculos pequenos.

Exemplo:


Resultado:

  • Arroz
  • Feijão
  • Carne

    • Lista ordenada HTML

    Uma lista ordenada inicia com a tag <ol>. Para classificar cada item da lista usa-se a tag <li>. Os itens dessa lista serão exibidos em sequência numérica como padrão.

    Exemplo:


    Resultado:

    1. Arroz
    2. Feijão
    3. Carne

    Lista de descrição HTML

    Sucintamente uma lista de descrição possibilita ao desenvolvedor web fazer uma lista com descrição de cada termo.

    Exemplo:

    Resultado:

    Arroz
    -Arroz branco normal
    Feijão
    -Feijão para a feijoada
