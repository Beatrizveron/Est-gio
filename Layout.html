Layout HTML

Normalmente os sites webs exibem o conteúdo em diversas colunas.

Elementos de Coluna:


Para definir as diversas partes/seções de uma página web, o HTML possui vários elementos semânticos que definem essas seções.


  • <header> Define o cabeçalho da página;
  • <nav> Define os links de navegação(menu da página);
  • <section> Define uma seção da página;
  • <article> Define um conteúdo específico;
  • <aside> Normalmente define uma barra lateral;
  • <footer> Define o rodapé da página.

    • Técnicas para criar Layout de várias colunas:


  • Grade CSS
  • Propriedade CSS float
  • CSS flexbox
  • Framework CSS
