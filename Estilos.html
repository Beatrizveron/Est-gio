Estilos de Texto HTML

Nessa seção disponibiliza-se o conteúdo sobre o atributo <style>. Esse elemento HTML é usado para definir estilos para um elemento, como fonte, cor, tamanho e etc…


Exemplo:

Texto em azul

Texto com tamanho Grande

Texto com a fonte courier.

Sintaxe:

<tag style = "propriedade:valor;">

Nota: propriedade e valor, ambos propriedade e valor CSS.


Cor de Fundo:

<body style = "background-color:powderblue;"> </body>

Cor do Texto:

<p style = "color:blue;"> Texto em azul </p>

Fonte do Texto:

<p style = "font-family:courier;"> Texto com a fonte courier </p>

Tamanho do Texto:

<p style = "font-size:200%;"> Texto com tamanho grande </p>

Alinhamento do Texto:

<p style = "text-align:center;"> Texto alinhado no centro </p>
