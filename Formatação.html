Formatação de Texto HTML

Caso queira definir o texto da sua página com uma formatação especial, o HTML contém diversos elementos para formatação de texto.

Elementos:

Esses elementos de formatação exibem vários tipos especiais de texto.


Tag Função Resultado
<b> </b> Negrito Negrito
<i> </i> Itálico Itálico
<em> </em> Para texto enfatizado Enfatizado
<sub> </sub> Para texto subscrito Textosubscrito
<sup> </sup> Para texto sobrescrito Textosobrescrito
<mark> </mark> Marcar texto Marcado
<strong> </strong> Para texto importante Importante
<small> </small> Para texto menor Texto menor
