Cores HTML

As cores HTML são caracterizadas com nomes de cores predefinidos ou com valores RGB, HEX, HSL, RGBA ou HSLA. O HTML suporta 140 nomes de cores padrão.

As cores em HTML podem ser especificadas utilizando o próprio nome da cor (no caso em inglês).

Cor de Fundo

A cor de fundo para elementos podem ser definidas como:

<h1 style="background-color:Blue;">Texto</h1>


<p style="background-color:green;">Texto</p>


Cor de Texto

A cor de texto para elementos podem ser definidas como:

<h1 style="color:red;">Texto</h1>


<p style="color:blue;">Texto</p>


Cor de Borda

A cor de borda para elementos podem ser definidas como:

<h1 style="border:2px solid red;">Texto</h1>


<h1 style="border:2px solid blue;">Texto</h1>
