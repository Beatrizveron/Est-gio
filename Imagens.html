Imagens HTML

As imagens são importantes quando se trata da estética visual de uma página web.

Sintaxe de Imagem

As imagens são vinculadas a páginas da web. A tag <img> é utilizada para vincular uma imagem em uma página da web. Essa tag cria um espaço de retenção para a imagem vinculada.

A tag <img> é vazia, possuindo apenas atributos e sem uma tag de fechamento.

Nesse contexto existem dois atributos obrigatórios:


  • src- para especificar o caminho para a imagem;
  • alt- para especificar a descrição da imagem (uma boa prática para leitores de tela).

    • <img src="url" alt="texto alternativo">

    Observação: uma página da web é carregada, é o navegador, naquele momento, que obtém a imagem de um servidor da web e a insere na página. Caso a imagem não estiver no mesmo lugar em relação à página da web e o navegador não conseguir encontrar a imagem. Os visitantes receberão um ícone de link quebrado e o “alt” com o texto alternativo.

    Largura e Altura da imagem

    Para fazer essas especificações o desenvolvedor da página pode utilizar o atributo <style>.

    Exemplo:


    <img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">


  • Width: Largura;
  • Heigth: Altura;

    • Nota:esses atributos definem a largura e altura em pixels por padrão.


    Imagem localizada em pasta

    Caso tiver suas imagens em uma subpasta, você deverá incluir o nome da pasta no atributo "src":


    <img src="pasta/url" alt="Texto alternativo" style="width:128px;height:128px;">

    Imagem com link

    Caso queira usar uma imagem como link, coloque a tag <img> dentro da tag <a>:

    <a href="#"> <img src="pasta/url" alt="Texto alternativo" style="width:128px;height:128px;"></a>


    Imagem localizada em outro servidor/site

    Caso você queira apontar para uma imagem em outro servidor, você deve especificar um URL absoluto (completo) no atributo src:

    <img src="https://url absoluto/images/imagem.jpg" alt="Texto alternativo">


    Imagem flutuante

    Para deixar a imagem flutuar à direita ou à esquerda de um texto use a propriedade CSS chamada "float:"

    <p>img src="url" alt="Texto alternativo" style="float:right;width:42px;height:42px;">Texto</p>
