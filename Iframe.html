Iframe HTML

Os iframes são comumente utilizados para exibir uma página web dentro de uma outra página.

Sintaxe:

A tag <iframe> classifica um quadro embutido e é responsável por incorporar outro documento em um documento do HTML.

<iframe src="url" title="descrição"></iframe>

Definir altura e largura do Iframe:

Os atributos height e width são utilizados para especificar tamanho e largura. Eles são especificados por pixels como padrão.

Exemplo 1:

<iframe src="url" height="200" width="300" title="descrição"></iframe>

Exemplo 2 com o atributo style:

<iframe src="url" style="height:200px;width:300px;" title="descrição"></iframe>


Remover borda de Iframe:

Por padrão o Iframe possui uma borda ao seu redor, para removê-la basta adicionar o <style> atributo e usar a <border> propriedade CSS.

Exemplo:

<iframe src="url" style="border:none;" title="descrição"></iframe>

Destino para link:

O iframe também pode ser usado como destino de um link. Para isso o atributo do link <target> deve se referir ao name atributo do iframe.

Exemplo:

<iframe src="url" name="iframe_a" title="descrição"></iframe>

Ou

<p><a href="url" target="iframe_a">Link.com</a></p>
